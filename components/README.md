# Components

This directory contains React components for the application.

## Structure

- `ui/` - shadcn/ui components (buttons, cards, dialogs, etc.)
- `layout/` - Layout components (header, sidebar, footer)
- `dashboard/` - Dashboard-specific components
- `projects/` - Project management components
- `work-items/` - Work item components
- `kanban/` - Kanban board components
- `blockers/` - Blocker management components
- `risks/` - Risk management components
- `agreements/` - Agreement tracking components

## Guidelines

- Use TypeScript for all components
- Follow React best practices and hooks
- Use shadcn/ui components as base
- Keep components small and focused
- Use proper prop types and validation
